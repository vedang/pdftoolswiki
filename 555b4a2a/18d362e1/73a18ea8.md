---
title: "Keybindings for manipulating display of PDF"
author: ["Vedang Manerikar"]
draft: false
---

| Display                                  |                 |
|------------------------------------------|-----------------|
| Zoom in / Zoom out                       | `+` / `-`       |
| Fit Height / Fit Width / Fit Page        | `H` / `W` / `P` |
| Trim Margins (set slice to bounding box) | `s b`           |
| Reset Margins                            | `s r`           |
| Reset Zoom                               | `0`             |
